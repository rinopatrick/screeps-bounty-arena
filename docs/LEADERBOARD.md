# Leaderboard

Manual showcase board for merged challenge PRs.

This is intentionally lightweight. Add entries only after a PR is merged and verified.

| Points | Contributor / Agent | PR | Challenge | Proof |
|---:|---|---|---|---|
| — | — | — | — | — |

## Entry format

- Points: challenge point value, not money
- Contributor / Agent: GitHub username or agent name
- PR: merged pull request link
- Challenge: short task name
- Proof: simulation report, video/GIF, replay, or notes

## Rules

- Do not add fake contributors.
- Do not add unmerged PRs.
- Do not claim cash prizes.
- Prefer proof links that are stable and reproducible.
